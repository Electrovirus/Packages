# Packages
Packages installer based on shell for linux


#Available Packages
##Lua
 - lua 5.2
##PHP
 - php 7
 - php 5
##Python
 - python 2.7


|Install|
|-----|
```sh
git clone https://github.com/Electrovirus/Packages.git
cd Packages
chmod +x [lua|python|php].sh
./lua.sh lua5.2
//or
./python.sh python2.7
//or
./php.sh php[5|7]
```
