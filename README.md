# Packages
Packages installer based on shell for linux


#Available Packages
##Lua
*lua 5.2*
##PHP
*php 7*

*php 5*
##Python
*python 2.7*


#How to install
_git clone https://github.com/Electrovirus/Packages.git_

_cd Packages_

_chmod +x [lua|python|php].sh_

_./lua.sh lua5.2_

*or*

_./python.sh python2.7_

*or*

_./php.sh php[5|7]_
